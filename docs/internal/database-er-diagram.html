职途安数据库 ER 图

实体关系模型（Entity-Relationship）- 内部设计视图

Internal Only: 本 ER 图用于研发/测试/运维协作，不对外公开部署。
PK: 主键 FK: 外键/关联键 1:N: 一对多关系 N:1: 多对一关系

users

  • PK user_id (UUID)
  • username (UNIQUE)
  • password_hash
  • role
  • created_at / updated_at

user_profiles

  • PK user_id
  • FK user_id -> users.user_id
  • school, major, graduation_year
  • expected_city, expected_salary
  • avatar_data_url, bio
  • online_seconds, ai_query_count ...

auth_sessions

  • PK session_id
  • FK user_id -> users.user_id
  • token_hash / token
  • expires_at
  • created_at / updated_at

ai_chat_messages

  • PK id
  • FK user_id -> users.user_id
  • session_id
  • role (user/bot)
  • content
  • created_at

forum_posts

  • PK post_id
  • FK user_id -> users.user_id
  • title, content
  • like_count, favorite_count, comment_count
  • created_at / updated_at

forum_post_comments

  • PK comment_id
  • FK post_id -> forum_posts.post_id
  • FK user_id -> users.user_id
  • parent_comment_id (root评论为空)
  • reply_to_user_id (二级回复目标)
  • content, created_at

forum_post_likes

  • PK id
  • FK post_id -> forum_posts.post_id
  • FK user_id -> users.user_id
  • created_at
  • 建议: UNIQUE(post_id, user_id)

forum_post_favorites

  • PK id
  • FK post_id -> forum_posts.post_id
  • FK user_id -> users.user_id
  • created_at
  • 建议: UNIQUE(post_id, user_id)

forum_post_attachments

  • PK attachment_id
  • FK post_id -> forum_posts.post_id
  • original_name, stored_name
  • content_type, file_size, file_path
  • created_at

analysis_reports

  • PK report_id
  • FK user_id -> users.user_id
  • business_type, input_summary
  • score, overall_risk_level
  • created_at

risk_items

  • PK id
  • FK report_id -> analysis_reports.report_id
  • matched_text, start_index, end_index
  • risk_level, reason, suggestion

forum_posts 1 : N forum_post_likes
forum_posts 1 : N forum_post_favorites
forum_posts 1 : N forum_post_attachments

users 1 : N analysis_reports
analysis_reports 1 : N risk_items

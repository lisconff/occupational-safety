职途安系统通信图

展示页面、接口、业务服务与外部依赖之间的通信链路

A. 前端页面通信入口

home.html

导航、入口跳转、登录态展示。

login.html

登录请求，token/userId/username缓存。

register.html

注册请求，成功后自动登录。

case.html

案例学习与模拟，行为上报。

ai-chat-test.html

AI问答与文件问答，历史记录同步。

forum/profile

社区互动、头像上传、雷达图渲染。

B. 通信协议与鉴权

HTTPS + JSON

前后端采用 RESTful 接口，统一响应结构 code/success/message/data。

JWT Bearer Token

登录后在请求头携带 Authorization，实现受保护接口访问控制。

C. 后端接口层

AuthController

/api/auth/register
/api/auth/login

UserController

/profile /avatar /activity /risk-assessment

ForumController

帖子、评论、点赞、收藏、附件下载。

AiAssistantController

AI问答、流式输出、历史会话。

D. 核心业务服务层

AuthService + JwtUtils

用户鉴权、密码校验、Token签发、会话持久化。

UserService

资料管理、头像处理、行为统计、风险评分计算。

ForumService + AiService

社区业务聚合、AI接口调用编排与返回封装。

E. 外部依赖与持久化

TiDB Cloud

用户、资料、会话、论坛、行为统计等核心数据持久化。

Coze API

后端通过 Token 调用 AI 能力，前端不直接暴露密钥。

Render 平台

前端静态托管 + 后端容器服务 + 健康检查与自动部署。

请求方向：前端 -> 接口 -> 服务 -> 数据/外部依赖 返回方向：统一 JSON 响应 -> 前端状态更新与图表渲染 关键闭环：行为上报 -> 风险评估 -> 个人中心雷达图
